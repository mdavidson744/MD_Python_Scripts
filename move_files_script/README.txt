Description:
simple python script to move files from one folder to another.
Upon moving the file, an output will be written.

To use:
1. Open python file in a code editor of your choice.
2. set source_folder & target_folder to the correct location.
3. Run move_files.py

Notes:
Be careful of the use of \ in folder paths. Use / or be careful to avoid escape character \ when you use it.
Use at own risk.
